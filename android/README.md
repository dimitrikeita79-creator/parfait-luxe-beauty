# Dossier Android (Capacitor)

Ce dossier sera **généré automatiquement** par Capacitor lors de la commande :

```bash
npx cap add android
```

Il contient le projet Android Studio natif (Gradle, `AndroidManifest.xml`, ressources).

Voir [`../EXPORT_ANDROID.md`](../EXPORT_ANDROID.md) pour la procédure complète.
